Description:
   This is an application to search and display videos by YouTube Data API.
Instructions:
   Users can enter any keyword to search the relevant videos and the videos found will be listed in either a list view or a grid view by pushing "list" or "grid" buttons.
   Users can filter the rated videos by the 5-stars rating control to find the videos that are above the seleted rating.
    Users can open a certain video by clicking its video thumbnail.

Other:
    Users can use "make run" in the Linux environment or java platform to compile and run the program.
    This program is implemented by Java SE8.
